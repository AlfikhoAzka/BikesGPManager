# BikesGPManager

BikesGPManager is a motorcycle racing management game inspired by the world of MotoGP. Build your own team, manage riders, develop bikes, handle finances, and compete against rival teams across multiple racing seasons.

The game focuses on strategy and team management instead of direct racing controls. Every decision matters, from signing young talents to upgrading bike performance and managing team budgets.

## Features

* Rider management system
* Team and staff management
* Bike development and upgrades
* Race weekend simulation
* Dynamic economy and finances
* AI-controlled rival teams
* Season progression system
* Randomized rider generation
* Save and load system
* Desktop support with Electron
* Modern UI built with React and Tailwind CSS
* Lightweight state management using Zustand

## Clone Repository

```bash
git clone https://github.com/AlfikhoAzka/BikesGPManager.git
cd BikesGPManager
```

## Project Setup

Create the React project using Vite:

```bash
npm create vite@latest motogp-manager -- --template react
cd motogp-manager
npm install
```

## Install Tailwind CSS

```bash
npm install tailwindcss @tailwindcss/vite
```

## Install Zustand

```bash
npm install zustand
```

## Install Electron

```bash
npm install electron electron-builder --save-dev
npm install concurrently --save-dev
```

## Install Library Countries

```bash
npm install i18n-iso-countries
```

## Run Development Server

```bash
npm run dev
```

## Build Project

```bash
npm run build
```
