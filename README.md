# BikesGP Manager

## Clone Repository

```bash
git clone https://github.com/AlfikhoAzka/BikesGPManager.git
cd BikesGPManager
```

## Setup Project

Buat project React menggunakan Vite:

```bash
npm create vite@latest motogp-manager -- --template react
cd motogp-manager
npm install
```

## Install Tailwind CSS

```bash
npm install tailwindcss @tailwindcss/vite
```

## Install Zustand

```bash
npm install zustand
```

## Install Electron

```bash
npm install electron electron-builder --save-dev
npm install concurrently --save-dev
```

## Menjalankan Project

```bash
npm run dev
```

## Build Project

```bash
npm run build
```
